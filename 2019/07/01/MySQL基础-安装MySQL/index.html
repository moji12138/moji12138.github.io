两种安装方式的区别

yum安装更为方便快捷，但定制性较差，比如配置文件、日志和数据库等存放位置都是固定的
源码编译安装比较繁琐，需要手动解决依赖，但更为灵活，在编译安装阶段即可指定安装、数据库、日志等位置
yum安装适用于定制性要求不高的个人或企业开发者
源码编译安装适用于定制性需求较高的个人或企业开发者
