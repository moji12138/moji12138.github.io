Linux基础 Vim文本编辑器

发表于 2019-05-31 | 分类于 Linux基础

Vim 常用操作

Vim 的几种模式

命令模式

光标定位

  • hjkl 左 上 下 右
  • 0 行首 $ 行尾
  • gg 首行 G 末行
  • nG 跳到第n行

查找-快速定位

  • /str 查找字符串str n找下一个 N回上一个
  • /^str 查找以str开头的行
  • /str$ 查找以str结尾的行

文本编辑

  • 复制 yy 3yy ygg yG (以行为单位)
  • 剪切(删除) dd 3dd dgg dG dw (以行为单位)
  • 粘贴 p(光标下一行) P(光标上一行)
  • 删除光标所在的字符 x
  • 从光标处删除到行尾 D 等同d$
  • 撤销 u
  • 重做 ^r
  • 修改一个字符 r

进入其它模式

  • A/a 行尾/光标后进入插入模式
  • O/o 上一行/下一行进入插入模式
  • I/i 行首/光标前进入插入模式
  • ESC 退出当前模式返回命令模式
  • : 跟命令 末行模式
  • v 可视化模式

插入模式

  • ^p 唯一补齐

末行模式

设置环境

  • set nu 显示行号
  • set nonu 取消显示行号
  • set tabstop=4 设置tab缩进字符数
  • set noh 取消高亮
  • :10 进入第10行

保存退出

  • :w 保存 (跟新文件名)另存为
  • :q 退出
  • :wq 保存并退出
  • :w! 强制保存
  • :q! 不保存并退出
  • :wq! 强制保存退出
  • :x 保存并退出(等同wq)
  • :wqa
  • :qa!

文件查找与替换

可视块模式

  • ^v 进入可视块模式
  • 块删除 选择块 d剪切 x删除
  • 块复制 选择块 y复制

快速注释或取消

  • 方法一：块选择模式

    注释

    1. 按ctrl+v 进入块选择模式 选择首字符
    2. 按大写I进入行首插入模式
    3. 输入注释符号 # //
    4. 按两下ESC注释选择的所有行

    取消注释

    1. 按ctrl+v 进入块选择模式 选择首字符
    2. 按大写I进入行首插入模式
    3. 删除注释符号 # //
    4. 按两下ESC注释选择的所有行

  • 方法二：替换命令

    ​ 注释

    ​ :10,20s/^/#/ 注释10到20行

    ​ :10,20s/^#// 取消10到20行的注释
