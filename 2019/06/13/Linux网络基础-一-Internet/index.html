发表于 2019-06-13 | 分类于 网络基础

什么是互联网

互联网(英语:Internet 音译:因特网)是数以亿万计的互连的计算设备（又称为主机）集合，它们通过通信链路连在一起，同时，通过交换网络实现分组交换。

互联网的各种物理设备

网卡(网络适配器)(NIC)

一般位于主机内部，一台想要接入网络的主机所必须配备的计算机硬件，每块网卡上都有一个MAC地址且是唯一的.MAC地址又叫物理地址，数据传输是通过物理地址来识别主机的。

集线器(HuB)

早期局域网中的基础设备，将多台计算机连接到一起，有数据发送的功能，发送数据时采用广播方式，不具有直接将数据发送到某节点的目的。

网桥(Bridge)

将两个相似网络连接起来，可对网络数据的流通进行管理。

交换机(Switch)

交换机内部的CPU会在每个端口成功连接时，通过将MAC地址和端口对应，形成一张MAC表。在今后的通讯中，发往该MAC地址的数据包将仅送往其对应的端口，而不是所有的端口。

路由器(Router)

路由器（Router）是连接因特网中各局域网、广域网的设备，它会根据信道的情况自动选择和设定路由，以最佳路径，按前后顺序发送信号。

网线

  • 同轴线缆
  • 双绞线
  • 光缆

互联网的接入方式

  • PSTN(拨号上网):互联网发展早期，网速慢，费用贵
  • ADSL:利用电话线接入互联网 速度在1Mbps-8Mbps
  • 光纤接入:利用光纤做传输介质，速率快，抗干扰能力强，一次性布线贵
  • 无线接入:利用无线射频进行通信，三大运营商的无线网络
  • HFC:利用有线电视网接入互联网
  • 电力网接入:利用低压电力线进行通信，如:电力猫

局域网 广域网 互联网

  • 局域网(LAN):由若干台计算机组成的小型网络，例如一个家庭、公司、学校中计算设备互联组成的网络。
  • 广域网(WAN):由无数个局域网组成的大型网络，例如一个城市、地区、国家中的局域网互联组成的网络。
  • 互联网:由全世界的广域网组成的全球计算机网络。

从A到B

  • 如果主机A和主机B在同一个局域网内，两台主机可能只需要经过一个交换机就可以相互通信了。
  • 如果主机A和主机B 在不同的局域网内，那么数据可能经过若干个交换机和路由器才能到达。
